# tbxt-hit-id - Original Development Repository

This is the original working repository used during development of the
TBXT hit identification pipeline, including the pre-hackathon design
and build phase and the hackathon weekend itself (TBXT Hit
Identification Hackathon 2026, Pillar VC, Boston).

For the cleaned, documented, and reproducibility-tested version, see:
**[`anandsahuofficial/tbxt-hit-id`](https://github.com/anandsahuofficial/tbxt-hit-id)**.

## What this repo is

This repository preserves the authentic development history: the
sequence of commits, code, scripts, and submission deliverables that
made up the work. It is intended as a public record of the work
rather than a turnkey reproduction package.

The earliest commits predate the hackathon weekend; the final commits
land at the submission deadline. Every commit timestamp is preserved.

## What this repo is not

- Not the recommended starting point for using or extending the
  pipeline. Use the polished repo for that.
- May contain experimental code and intermediate artifacts that did
  not make it into the final version.

## Citation

If you cite the pipeline, cite the polished repo
([`tbxt-hit-id`](https://github.com/anandsahuofficial/tbxt-hit-id))
rather than this one. This repo exists as timeline evidence.
